# 使用手册

### 集成技术
- rust ffi(动态库)与node通讯
- electron集成bootstrap样式库

### 根据流程运行项目
> 1、 cd new-app/

> 2、 npm install

> 3、 npm start

### 打包命令
> npm run packager

### blibli时间
完成工具，可以快乐的用electron开发客户端了，如果你熟悉rust语言，可以作为业务逻辑处理，前端用electron做界面；如果你不熟悉rust，直接用electron与系统API通讯也是可以的。
